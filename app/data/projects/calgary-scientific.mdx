from anywhere via a web browser or mobile app. This was the original
ResolutionMD work, years before I returned as a consultant.

The stack was a fair distance from a single language: C++ and Java with Spring
on the server, Flex / Silverlight / HTML5 on the web clients, and native
Android and iOS on mobile.
